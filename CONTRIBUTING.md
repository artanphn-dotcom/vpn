# Contributing

Thank you for considering contributing to FortiGate IPsec Configuration Generator.

How to contribute
- Fork the repository and create a feature branch.
- Open a pull request with a clear description and tests for new behavior.

Coding style
- Follow PEP8 / standard Python formatting. Projects use `black`/`flake8` where applicable.

Testing
- Add unit tests under `tests/` and ensure they pass locally with `pytest`.

Security
- Do not commit secrets (PSKs) into the repository. Use environment variables or local config files when testing.

License
- Add a LICENSE file before distributing publicly.
